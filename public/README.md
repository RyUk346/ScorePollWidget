# public/

Files here are served as-is at the site root (under your base path).

## fifa_2026.png

Put your FIFA World Cup 2026 logo image here, named exactly **`fifa_2026.png`**.

The intro slide loads it from `<base>/fifa_2026.png` (e.g.
`https://location.hyperglow.co.uk/hg_score_poll/fifa_2026.png`). If the file is
missing, the slide falls back to a text "WORLD CUP 2026" lockup automatically.

A transparent-background PNG looks best on the glass widget. Recommended size:
roughly 320×280 px (it's scaled to fit ~140 px tall).

> Use a logo you have the rights to display — the official FIFA emblem is a
> registered trademark.
